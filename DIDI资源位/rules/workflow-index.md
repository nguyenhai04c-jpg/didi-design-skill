# 工作流索引
> DiDi 司机端视觉产出 · 场景索引与映射规则

## 工作流程

```
Step 0  读取 rules/brand-rules.md（品牌总则，每次必读）
Step 1  确认场景 → 查下方「场景索引」找到对应 reference 文件
Step 2  Read 对应 reference/*.md（规范细则）
Step 3  产出图片，输出至 outputs/
```

## 文件夹说明

| 文件夹 | 用途 |
|-------|------|
| `rules/` | 指导规则（品牌总则 + 本索引），每次产出前读取 |
| `reference/` | 各场景规范细则，按需读取 |
| `logos/` | 顶部 / 底部 logo 素材 |
| `fonts/` | 滴滴活力黑字体（开屏版式二） |
| `images/` | 各尺寸底部 logo 图片 |
| `outputs/` | 产出图片归档 |
| `素材/` | 临时存放当次工作的原始图片素材 |
