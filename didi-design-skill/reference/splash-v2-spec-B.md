## 主标题设计原则

1. **色彩**：严格遵循色彩系统规范，因背景色彩不同，规范仅可使用滴滴桔、深灰色、白色或黑色
   - **默认字色映射**：浅色背景 → 黑色 `#000000`；深色背景 → 白色 `#FFFFFF`
   - 如需强调品牌感可使用滴滴桔 `#FF6400`，需人工指定
2. **字体**：主标题中文字体为滴滴活力黑体，须遵循示例，倾斜角度为向右和向上 5°
3. **数字**：主标题中的数字应用字体：滴滴活力黑体
4. **位置**：在正常情况下应保持在文字编辑区域内上下居中对齐

## 图片合成注意事项

- 主图按图片编辑区宽度等比缩放，垂直方向**居中对齐**于图片编辑区
- 若画面尺寸小于规范尺寸，则等比放大画面
- 素材图铺满全画布，文案叠在上面

```python
img_area_top    = 864
img_area_bottom = 1908
img_area_height = img_area_bottom - img_area_top  # 1044 px

scale = canvas_width / bg.width
bg_resized = bg.resize((canvas_width, int(bg.height * scale)))

# 垂直居中于图片编辑区
offset_y = img_area_top + (img_area_height - bg_resized.height) // 2
canvas.paste(bg_resized, (0, offset_y))
```
