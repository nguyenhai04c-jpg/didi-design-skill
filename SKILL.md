# Didi Design Skill

视觉设计 Skill Framework

## Purpose

用于滴滴运营设计资源位延展、Banner设计、Splash设计、品牌视觉规范执行。

## Skill Priority

执行顺序：

1. rules
2. reference
3. fonts
4. logos
5. images
6. outputs

## Output Rules

- 严格遵循对应资源位尺寸
- 不允许自由调整版式
- 优先遵循 Banner Skill
- 优先遵循 Splash Skill
- 输出前检查字体、间距、Logo安全区
